# Deployment Guide

## Building for Production

To build your site for production:

```bash
npm run docs:build
```

## Deployment Options

1. **GitHub Pages**
   - Perfect for documentation sites
   - Free hosting
   - Automatic deployments

2. **Netlify**
   - Easy setup
   - Great performance
   - Continuous deployment

3. **Vercel**
   - Zero configuration
   - Global CDN
   - Automatic HTTPS
